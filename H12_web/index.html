Summary: Weather Analysis



In this project, we created a Python script to visualize the weather of 500+ cities across the world of varying distance from the equator. To accomplish this, we used a simple Python library, the OpenWeatherMap API, and a little common sense to create a representative model of weather across world cities.


Lastly, we built a series of scatter plots using matplotlib to showcase the relationships between temperature (F), humidity (%), wind speed (mph), and latitude. This site provides the source data and visualizations created as part of our analysis, as well as explanations and descriptions of any trends and correlations witnessed.
